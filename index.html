BRIEF HISTORY ON THIS PROJECT

This Project start in december 12th 2022, it was an idea i've kinda never seen before, so as i was learning Unity I thought it would be a good idea to learn on the way by developing my idea. So little by little and with all the effort i've put till now, the idea has started to form, with also the support of other devs who i met on the way and wanted to make this project a real thing.
